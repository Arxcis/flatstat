This site is a gift to all the hard-working flatpak and portal maintainers. You are the coolest developers on the planet 🪐

The data is sourced from repos belonging to https://github.com/flathub. In each repo, we look for the APP_ID.(json|yaml|yml)-file. To get historic data, we use Github Graphql API.

Source code can be found here https://github.com/Arxcis/flatstat/
